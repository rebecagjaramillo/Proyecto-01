# CV Web en HTML

## Propósito del Proyecto

Este proyecto consiste en una página web básica que funciona como un **Currículum Vitae digital**, desarrollada únicamente con **HTML**.

El objetivo del proyecto fue practicar los fundamentos del desarrollo web, entendiendo cómo estructurar una página utilizando etiquetas HTML y organizar información personal y profesional dentro de una página web.

Además, este proyecto sirvió como práctica para aprender a utilizar **Git** y publicar un repositorio por primera vez en **GitHub**, aplicando comandos básicos de control de versiones.

La página incluye secciones típicas de un CV como:

* Perfil profesional
* Experiencia
* Educación
* Habilidades
* Datos de contacto

Este proyecto representa una introducción al desarrollo web y al uso de herramientas utilizadas en entornos profesionales de programación.

---

## Instrucciones para Ejecutar el Proyecto

### Requisitos

Para ejecutar este proyecto solo necesitas:

* Un navegador web (Chrome, Edge, Firefox, etc.)
* Tener descargado o clonado el repositorio

### Pasos para ejecutar

1. Clonar el repositorio:

```
git clone https://github.com/tu-usuario/tu-repositorio.git
```

2. Entrar a la carpeta del proyecto.

3. Abrir el archivo `index.html` en cualquier navegador web.

También puedes abrirlo directamente haciendo doble clic en el archivo.
